# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. All contributors, maintainers, and participants are
expected to uphold a harassment-free, welcoming environment regardless of
background, identity, or level of experience.

**Scope.** This code of conduct applies within all project spaces (repository,
issues, pull requests, discussions) and in any public space where someone is
representing the project.

**Enforcement.** Concerns can be reported to
[an0sunshy@outlook.com](mailto:an0sunshy@outlook.com). All reports will be
reviewed promptly and handled confidentially. The full enforcement guidelines
are defined in the Contributor Covenant text linked above.
